# Personal Portfolio Website

## Project Description
This project is a responsive personal portfolio website created for Assignment 1 (SWE363).  
It is built using HTML, CSS, and JavaScript and includes interactive features and documentation.

## Features
- Responsive design (desktop, tablet, mobile)
- Dark/Light mode toggle
- Smooth scrolling navigation
- Contact form interaction (no backend required)
- Organized project structure

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub for version control and deployment
